

## Prerequisites

* Install [Node.js](https://nodejs.org/en/download/)

* Install TypeScript

```bash
npm install -g typescript
```

#### Installing dependencies

```bash
cd <redux-store-project>

yarn install
# OR
npm install
```

#### Local server

```bash
yarn start
# OR
npm start
```

Visit [localhost:8000](localhost:8000) in your browser.
